# flutter_torico_id_login

Flutter TORICO-ID Login Plugin.  

## Requirements
- Dart sdk: ">= 2.7.0 < 3.0.0"
- Flutter: ">=1.10.0"
- Android: minSdkVersion 17 and add support for androidx
- iOS: --ios-language swift, Xcode version >= 9
